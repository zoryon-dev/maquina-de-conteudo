## Variáveis de Substituição

| Variável | Descrição | Exemplo |
|----------|-----------|---------|
| `{{title}}` | Título principal | "5 Dicas de Marketing" |
| `{{headline}}` | Headline | "O Segredo do Sucesso" |
| `{{content}}` | Conteúdo do corpo | Texto completo |
| `{{descricao}}` | Descrição | Texto de apoio |
| `{{subtitulo}}` | Subtítulo | "Um guia completo" |
| `{{legenda}}` | Legenda | "Para quem quer aprender" |
| `{{paragrafo1}}` | Primeiro parágrafo | "Introdução..." |
| `{{paragrafo2}}` | Segundo parágrafo | "Desenvolvimento..." |
| `{{destaque}}` | Destaque | "⚡ Importante" |
| `{{cta}}` | Call-to-action | "Saiba mais" |
| `{{author}}` | Autor | "@nome" |
| `{{date}}` | Data | "Jan 2026" |
